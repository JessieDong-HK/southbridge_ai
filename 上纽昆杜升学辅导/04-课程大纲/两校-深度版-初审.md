# 两校 · 深度版 · 初审阶段大纲

> 适用：同时申请上海纽约大学 + 昆山杜克大学，选择保入围深度陪跑版（¥49,800）

## 服务配置

- **教学形式**：全部 1v1
- **参与人员**：核心主讲老师 + Cheryl + 督学小助手（陪伴式贴心学姐）
- **文书+AI面试双线并行**

---

## 课程流程

### 师生破冰见面会 + 时间线梳理（30min）

参与：核心主讲老师 + Cheryl + 学生（家长可旁听）

1. 两位老师自我介绍 + 学生自我介绍（10min）
2. 上纽 & 昆杜官方申请与考核时间线（5min）
3. 我们给学生规划的时间线（5min）
4. 强调全程陪伴："不只是上课，更会帮你抓机会、疏解情绪"（5min）
5. Q&A + 约定下次故事挖掘课

### 3个故事深度挖掘（60min）

授课：文书老师 + 昆杜AI面试老师协同

1. 软实力故事（20min）：领导力/主动性/问题解决/同理心
2. 硬实力故事（20min）：学科能力/学习力/专业潜质
3. 专业方向锁定+专业故事（20min）
4. 上纽文书故事线梳理（15min）

### 文书故事线沟通（30min）

结合三大故事，梳理适配上纽文书的核心故事线。

### 文书定稿沟通（30min）

带学生读一遍文书，确认理解内容、理解好在哪里、修改意见讨论。

### 全套申请材料终审会（30min）

逐项核对全套申请材料信息。

---

## AI面试课程（4节 1v1）

### 第1课：STAR框架 + 故事结构化 + 套题 + 追问（90min）

1. 昆杜AI面试核心规则 + AI评分4维度解析（10min）
2. STAR框架讲解（5min）
3. 三大故事STAR结构化打磨 + 英文练习纠错（35min）
4. 套题训练：5道高频真题（15min）
5. 追问模拟训练：每故事2-4个细节追问点（15min）
6. 课堂总结+Key Takeaway（5min）

**作业**：三大故事STAR中英文撰写（英文2.5-3分钟），脱稿练习

### 第2课：作业复盘 + 昆杜关键词 + 专属资源 + 实战（60min）

课前老师准备：
- 批改作业，润色英文表达
- 准备2个与学生绑定的昆杜专属资源
- 准备昆杜5大核心关键词及案例
- 准备5道高频真题

1. 作业复盘（10min）
2. 昆杜5大核心关键词+案例讲解+即时抽查（10min）
3. 专属资源匹配讲解（15min）
4. 3道真题问答+逐题纠错（20min）
5. 学生总结+Takeaway（5min）

### 第3课：社会思辨话题（60min）

1. 上节课作业复盘（10min）
2. 思辨题通用高分结构讲解（5min）
3. 高频思辨真题精讲+现场练习（40min）：AI伦理、全球问题、气候变化、教育公平、科技与社会、个人责任与集体
4. 学生总结+Takeaway（5min）

**AI模考双方案**：自主模考平台（4次机会）或 真人模拟AI面试（1次）

### 第4课：模考复盘 + 薄弱项冲刺 + 考前调整（60min）

1. 模考全面复盘（15min）
2. 个性化薄弱项专项强化（30min）
3. 考前全流程核对+避雷清单（5min）
4. 课程总结+考前信心建立（10min）
