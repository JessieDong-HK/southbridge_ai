# 文书辅导

## 所属阶段

初审阶段 · 上海纽约大学核心环节

## 服务目标

通过系统化的故事挖掘和文书打磨流程，帮助学生建立清晰的申请画像，产出招生官一眼记住的个人故事。

---

## 辅导流程

### 深度版全流程

```
师生破冰见面会（30min）
    ↓ Cheryl + 核心主讲老师 + 学生
3个故事深度挖掘（60min）
    ↓ 文书老师 + AI面试老师协同
文书故事线沟通（30min）
    ↓
文书撰写（学生 + 老师反馈）
    ↓
文书定稿沟通（30min）
    ↓
全套申请材料终审会（30min）
```

### 标准版流程

```
督学小助手破冰见面会（30min）
    ↓
故事挖掘+文书故事线梳理（60min）
    ↓
文书撰写（学生 + 群内反馈）
    ↓

```

## 第1步：破冰见面会（30min）

**目标**：建立信任、消除陌生感、讲清申请节奏

- 老师自我介绍 + 学生自我介绍（10min）
- 上纽 & 昆杜官方申请与考核时间线（5min）
- 我们给学生规划的时间线（5min）
- 强调全程陪伴："不只是上课，更会帮你抓机会、疏解情绪"（5min）
- Q&A + 约定下次故事挖掘课

## 第2步：3个故事深度挖掘（60min·深度版）

**授课人员**：文书老师 + 昆杜AI面试老师协同

这是整个文书辅导的核心环节。不是帮学生"写作"，而是帮学生**发现和重构**自己的经历。

### 1. 软实力故事（20min）
聚焦：社团/班级工作、志愿者服务、团队协作、冲突解决、主动担当、助人成长、责任类经历

→ 锁定 **1个最有细节、有波折、能体现领导力/主动性/问题解决/同理心** 的核心经历

### 2. 硬实力故事（20min）
聚焦：学科竞赛、科创项目、学术探索、文体赛事、系统性学习突破、目标达成类成就

→ 如无竞赛经历，转化为学业成长、自主学习、长期坚持、能力突破类高光
→ 锁定 **1个最能体现学科能力/学习力/专注力/专业潜质** 的核心经历

### 3. 专业方向锁定 + 专业故事（20min）
- 若学生有明确专业+真实经历支撑：直接梳理专业适配故事
- 若学生专业迷茫：老师结合学生特质现场定制适配专业 → 搭建专业适配故事

### 4. 上纽文书故事线（15min）
结合挖掘的三大故事，梳理适配上海纽约大学文书题目的核心故事线

## 第3步：文书故事线沟通（30min）

深度版独立环节。结合挖掘的三大故事，确认文书的核心故事线和写作方向。

## 第4步：文书定稿沟通（30min）

带着学生读一遍完成的文书：
- 确认学生清晰理解文书内容
- 让学生理解文书好在哪里
- 讨论修改意见
- 确保最终版本学生完全认同

## 第5步：全套申请材料终审会（30min）

带领学生完整过一遍全套申请材料，逐项核对信息，确保无误提交。

## 核心理念

> 不是增加经历，而是帮学生建立一个清晰、统一、有说服力的申请画像。

大多数高考生的共性问题："每天都在刷题和考试，没什么经历可写。"南桥的解决方式不是帮学生编造经历，而是从已有经历中提炼出与学校匹配的故事。

## 招生官视角

招生官不是在找"做最多事的人"，而是在找**与学校文化高度匹配、有独立思考、能清晰表达自己的人**。

南桥文书的核心优势：由真正经历过选拔的学长学姐指导——Cheryl学姐曾担任上纽官方文书辅导助理，累计深度辅导200+份文书。

## 典型案例：Bridge Builder 文书重构

### 学生初始思路
"我在博物馆做过志愿者，负责讲解，接待过外国游客，因此我是传播中国文化的bridge。"

→ 问题：经历平淡，缺乏亮点，难以体现个人能力与价值

### 辅导后的重构
聚焦一个具体场景（接待意大利高中生），描述具体的观察和行动：
- 发现传统讲解无效 → 主动调整为"讲故事"方式
- 设计中意武器对比互动 → 激发主动思考
- 方法分享给团队 → 优化整体讲解方式

→ 结果：顺利进入复审，最终拿到A档

### 招生官会看到什么
"这不仅是一段志愿者经历，更是一个能在跨文化场景中观察问题、调整表达、建立连接，并影响他人的学生。"

---

## 注意事项

- **严禁代写** — 角色是引导和打磨，最终文书必须出自学生本人
- **学生需高度配合** — 提供真实经历、积极参与讨论
- **两校文书有差异** — 上纽注重个人故事和匹配度，需针对性调整
- **早规划效果更好** — 理想情况下高一/高二开始规划，时间更充裕
