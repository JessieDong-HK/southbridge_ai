# 07 工作流程 · 督学小助手工作流程

## 角色定位

**陪伴式贴心学姐**（深度版专属配置）

不只是课程记录者、作业提醒者，更是学生在整个申请周期里**亲近信任、敢说心里话的学姐**。全程真诚陪伴、主动关心、信息同步、情绪支撑，搭建学生 ↔ 核心老师之间最顺畅的桥梁，让学生全程有归属感、安全感、被重视感。

> 标准版的"小助手"角色偏教务+浅度陪伴（课程笔记+节点提醒+1次心理疏导），本文档描述的是深度版督学小助手的完整工作内容。

---

## 一、基础本职工作

### 每节课后
1. 及时整理腾讯会议转录文字
2. 输出完整版课程笔记
3. 同步发送至学生及家长群
4. 同步更新内部学员进度档案，记录学习情况、作业提交状态及薄弱项

### 作业管理
- **分层提醒**：截止前5天首次提醒 → 截止前24小时二次提醒
- **未提交学员持续督促**，确保所有学员按时完成
- 收集作业、整理资料
- 对接模考安排、预约会议

### 档案维护
- 维护学员档案
- 全程跟踪学生状态、薄弱点、进度变化

---

## 二、贴心学姐专属陪伴服务（核心）

### 固定轻量聊天式沟通

全程至少安排 **2-3 次 15-20 分钟轻松聊天会**：
- 节奏宽松、氛围像朋友
- **不讲课、不留作业、不考核**
- 只聊真实状态

### 每次聊天的话题

| 方向 | 具体聊什么 |
|------|------|
| 心态 | 最近学习累不累、有没有焦虑、有没有内耗 |
| 状态 | 近期时间安排、夏校/活动/考试计划 |
| 困惑 | 对课程、申请、未来的迷茫 |
| 心事 | 不敢跟家长说的想法和顾虑 |
| 情绪 | 任何小情绪、小压力、小纠结都可以放心说 |

### 关键信息捕捉

主动倾听、细心观察，学生随口提到的信息**全部记牢**：
- 参加夏校（DKU / 上纽）
- 竞赛、活动、考试
- 家庭安排
- 心态波动、目标变化
- 顾虑担忧

**一旦发现学生有机会、有困惑、有需求、有风险 → 立刻同步给核心老师**，由核心团队针对性安排会议、给出建议、提供支持。

### 情绪陪伴与心理支撑

- 学生焦虑、拖延、没信心、自我怀疑时 → **第一时间安抚、鼓励、共情**
- **不说教、不施压**，站在学生角度理解压力
- 做**情绪树洞**
- 申请季全程持续正向反馈，让学生始终觉得 "有人陪着我、有人懂我、有人支持我"

### 信息前置科普与机会提醒

主动提醒学生不知道的重要信息：
- 夏校该做什么、能争取什么资源
- 如何和教授互动、如何拿到推荐信
- 申请节点、材料细节、面试注意点
- 两校偏好、申请小技巧

**用学姐的视角轻松分享**，不生硬、不说教，让学生少走弯路。

### 建立长期真诚亲近的关系

- 用平等、轻松、真诚的语气沟通，**不端着、不形式化**
- **记住学生的喜好、性格、状态、小目标**，让学生感受到 "被认真记住、被真心在乎"
- 全程保持稳定、可靠、温暖的存在感，让学生愿意主动分享、主动求助

---

## 三、执行节奏

| 时间节点 | 行动 |
|------|------|
| **相识破冰会后** | 学姐第一次轻聊，建立初步熟悉感 |
| **故事挖掘课 + 文书阶段** | 每周轻聊一次，跟进状态、收集信息 |
| **面试课程密集期** | 随时关心学习压力、作业进度、心态变化 |
| **夏校 / 关键活动前** | 专门沟通一次，梳理机会、提醒重点、同步核心老师 |
| **材料复审 & 申请递交前** | 安抚焦虑、打气鼓励、全程陪伴 |

---

## 四、最终效果

- 学生不会觉得自己只是 "被上课、被交付服务"
- 全程有**亲近的学姐陪着、懂他、支持他、帮他搭桥**
- 所有机会、信息、困惑都能被及时接住，不会错过任何优势
- 师生关系极强、信任感拉满、情绪稳定、申请状态更好
- 整个服务从 **"课程教学"** 升级成 **真诚陪伴式高端升学支持**
