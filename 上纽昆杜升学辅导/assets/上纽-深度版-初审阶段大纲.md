**参与人员**：核心主讲老师 \+ Cheryl \+ 学生（家长可旁听）

简单说明今天 30 分钟会做什么：认识彼此→\\两校申请时间线→我们一起的准备节奏

10min 两位老师自我介绍\+学生自我介绍 

5min 上纽 \& 昆杜 官方申请与考核时间线

5min 我们给学生规划的时间线

5min强调：整个申请季我们会全程陪着你，不只是上课，更会帮你抓机会、疏解情绪。有任何事都可以跟我们说，我们一起解决。

# 3个故事深度挖掘（60min）

**授课人员**：文书老师 \+ 昆杜AI面试老师 协同授课

1. **20min 软实力故事深度挖掘\+细节补充成型**：聚焦社团/班级工作、志愿者服务、团队协作、冲突解决、主动担当、助人成长、责任类经历，最终锁定 **1个最有细节、有波折、能体现领导力、主动性、问题解决能力、同理心** 的核心经历，梳理完整事件脉络。

2. **20min 硬实力故事深度挖掘\+细节补充成型**：聚焦学科竞赛、科创项目、学术探索、文体赛事、系统性学习突破、目标达成类成就；无竞赛经历则转化为学业成长、自主学习、长期坚持、能力突破类高光，最终锁定 **1个最能体现学科能力、学习力、专注力、专业潜质** 的核心经历。

3. **20min 专业方向锁定 \+ 专业故事深度挖掘\+细节补充成型**：

    1. 若学生有明确专业\+真实经历支撑：直接梳理专业适配故事，强化与专业的关联性

    2. 若学生专业迷茫、无明确方向：老师结合学生特质现场定制适配专业，经学生确认后，搭建专业适配故事

4. **15min 上纽文书故事线**：结合挖掘的三大故事，梳理适配上纽文书的核心故事线

# 第1课：STAR 框架 \+ 三大故事结构化 \+ 套题训练 \+ 追问实战（90min）

**本节课核心目标**：掌握STAR框架用法，实现三大故事结构化，学会套题复用技巧，提升追问应对能力

1. **10min 昆杜AI面试核心解析**：讲解昆杜AI面试核心规则（双机位、答题时限、二刷要求等）；解析AI评分4大核心维度（内容完整性、逻辑连贯性、英语表达、院校匹配度）

2. **5min STAR框架讲解**：简洁拆解S（情境）、T（任务）、A（行动）、R（结果\+反思），明确各模块高分重点

3. **35min 三大故事STAR结构化打磨**：将三大故事逐一套入STAR框架，学生练习英文表述，现场纠正表达问题

4. **15min 套题训练**：讲解“只改开头结尾、不变核心故事”的套题方法，结合5道高频真题开展套题实战，强化复用能力

5. **15min 课前预设追问模拟训练**：老师提前根据学生三大故事，为每个故事准备2\-4个细节追问点，追问方向聚焦：具体行动步骤、当时的真实想法、遇到的最大困难、如何说服他人、后续改进措施、反思总结等

6. **5min 课堂总结 \+ Key Takeaway**：梳理本节课核心知识点，强调三大故事、STAR框架、套题技巧的重点应用

**第1课课后作业**

1. 完成三大故事的STAR结构化撰写（中英文完整版），英文表述控制在2\.5–3分钟，熟练脱稿练习

2. 下次课前24小时提交作业，老师必须在第2节课前完成全部批改，标注问题及优化方向

# 第2课：作业复盘 \+ 昆杜关键词 \+ 专属资源匹配 \+ 实战练习\&即时纠错（60min）

**课前老师准备**

1. 批改完第1课三大故事STAR中英文作业，标注逻辑、表达等问题，给出优化方向，润色英文表述（纠正错误及Chinglish）

2. 提前准备2个与学生专业、三大故事强绑定的昆杜专属资源（例如：实验室、研究中心、教授项目、RA机会、Study Away交换校区、特色社团、学生岗位、实践项目）

3. 准备昆杜5大核心关键词及对应极简案例，用于课堂讲解

4. 准备5道高频真题，用于课堂现场问答练习

**第2课 课堂流程**

1. **10min 第1课作业复盘**：集中反馈作业批改情况，针对性纠正学生在STAR框架应用、英文表达上的问题

2. **10min 昆杜5大核心关键词\+案例讲解**：

    - ① **Liberal Arts（通识博雅教育）** 案例：昆杜通过跨领域通识课，帮学生搭建完整知识框架，不局限于单一专业

    - ② **Interdisciplinary（跨学科）** 案例：学生结合不同学科知识完成课题、项目，打破专业边界

    - ③ **Global Campus \& Study Away（全球校园与海外交换）** 案例：昆杜学生大三可前往NYU全球各校区交换，衔接全球优质教育资源，拓展国际视野

    - ④ **Small Class \& Mentorship（小班教学与导师制）** 案例：小班授课保障师生充分互动，一对一导师为学生提供学术规划、科研指导等个性化支持

    - ⑤ **Community \& Social Impact（校园社群与社会责任感）** 案例：学生通过社团活动、实践项目、公益行动，创造价值、融入校园社群、回馈社会

    - 课堂即时抽查：学生关键词记忆及案例掌握情况

3. **15min 专属资源匹配讲解**：基于学生的三大故事及专业方向，讲解提前准备的2个昆杜专属资源，明确资源与学生的适配点，指导学生融入面试回答

4. **20min 3道真题问答 \+ 逐题即时纠错**：

5. **5min 学生总结 \+ Takeaway**：让学生梳理本节课重点，强化关键词、资源匹配、真题答题技巧的记忆

**第2课课后作业**

1. 完成课上练习的5道真题中英文回答，结合老师纠错建议优化完善

2. 下次课前24小时提交作业，老师提前批改反馈

# 第3课：社会思辨话题（60min）

1. **10min 上节课作业复盘**

2. **5min 思辨题通用高分结构讲解**

3. **40min 高频思辨真题精讲\+现场练习**：讲解3个昆杜高频思辨真题（例如：AI伦理、全球问题、气候变化、教育公平、科技与社会、个人责任与集体），每个真题给出完整范例后，让学生现场口述回答，老师即时纠错、优化

4. **5min 学生总结\+Takeaway**

**第3课课后作业**

1. 完成课上练习的3道思辨真题中英文回答，结合老师纠错建议优化

2. 完成AI模考（按以下双方案执行）

3. 下次课前24小时提交作业及模考相关材料

**AI模考双方案**

- **方案A：自主模考平台已开发完成（最优方案）**

    1. 学生在平台拥有4次全真AI模拟面试机会，自主安排时间完成

    2. 学生从4次模拟中自选1次最满意的版本提交

    3. 平台生成完整匹配度报告，老师结合报告进行全面复盘

- **方案B：模考平台未上线（兜底标准方案）**

    1. 为学生安排1次一对一真人模拟AI面试，完全还原昆杜真题、计时规则、双机位要求、追问逻辑

    2. 面试结束后，老师现场给出完整点评及优化建议

**统一提交要求**：无论执行A/B方案，学生均需在第4课课前24小时，提交最终选定的模拟面试视频/记录，老师提前完成完整复盘准备。

# 第4课：模考复盘 \+ 薄弱项专项冲刺 \+ 考前全流程与心态调整（60min）

1. **15min 模考全面复盘**：逐维度点评模考表现，重点分析内容完整性、逻辑连贯性、英语表达、答题节奏、追问应对、超时/偏题等问题

2. **30min 个性化薄弱项专项强化**    

3. **5min 考前全流程核对 \+ 避雷清单**

    - 设备核对：双机位设备、网络、光线、背景、着装要求

    - 规则核对：答题时限、节奏把控、二刷规则

4. **10min 全程课程总结 \+ 考前信心建立**：梳理全课程核心知识点，强化学生答题信心，讲解临场应变技巧

# 通用课后服务

1. 课程结束后，及时整理腾讯会议转录文字，输出完整版课程笔记，同步发送至学生及家长群

3. 作业分层提醒：截止前5天首次提醒、截止前24小时二次提醒；对未提交学员持续督促，确保所有学员按时完成作业

## 二、课后老师工作

学员按规定时间提交作业后，老师提前完成逐字精批，明确标注问题、给出具体优化建议及可落地的修改方向，确保学员针对性提升。













# 督学小助手定位：陪伴式贴心学姐

核心定位

不只是**课程记录者、作业提醒者**，更是学生在整个申请周期里**亲近信任、敢说心里话的学姐**，全程真诚陪伴、主动关心、信息同步、情绪支撑，搭建学生 ↔ 核心老师之间最顺畅的桥梁，让学生全程有归属感、安全感、被重视感。

一、基础本职工作

每节课后整理会议转录、输出完整课程笔记，同步学生、家长、核心老师

记录学员进度、作业提交情况，按时提醒（提前 5 天→提前 24 小时）

收集作业、整理资料、对接模考、预约会议、同步信息

维护学员档案，全程跟踪学生状态、薄弱点、进度变化

二、升级新增：贴心学姐专属陪伴服务（核心新增）

固定轻量聊天式沟通（非上课、无压力、纯谈心）

全程至少安排 **2–3 次 15–20 分钟轻松聊天会**，节奏宽松、氛围像朋友

不讲课、不留作业、不考核，只聊真实状态：

最近学习累不累、心态怎么样、有没有焦虑、有没有内耗

近期时间安排、夏校 / 活动 / 考试计划、个人想法和困惑

对课程、对申请、对未来有没有迷茫、不敢跟家长说的心事

任何小情绪、小压力、小纠结都可以放心说

主动捕捉学生关键信息，第一时间同步核心老师

主动倾听、细心观察，学生随口提到的信息全部记牢：比如：参加夏校（DKU / 上纽）、竞赛、活动、考试、家庭安排、心态波动、目标变化、顾虑担忧

一旦发现学生**有机会、有困惑、有需求、有风险**，立刻同步给核心老师

由核心团队针对性安排会议、给出建议、提供支持，不让学生自己盲目摸索

情绪陪伴与心理支撑

学生焦虑、拖延、没信心、自我怀疑时，第一时间安抚、鼓励、共情

不说教、不施压，站在学生角度理解压力，做 情绪树洞

申请季全程持续正向反馈，让学生始终觉得 “有人陪着我、有人懂我、有人支持我”

信息前置科普与机会提醒

学生不知道的重要信息主动提醒：夏校该做什么、能争取什么资源、如何和教授互动、如何拿到推荐信申请节点、材料细节、面试注意点、两校偏好、申请小技巧

用学姐的视角轻松分享，不生硬、不说教，让学生少走弯路

建立长期真诚亲近的关系

用平等、轻松、真诚的语气沟通，不端着、不形式化

记住学生的喜好、性格、状态、小目标，让学生感受到 “被认真记住、被真心在乎”

全程保持稳定、可靠、温暖的存在感，让学生愿意主动分享、主动求助

三、执行节奏

**相识破冰会后**：学姐第一次轻聊，建立初步熟悉感

**故事挖掘课 \+ 文书阶段**：每周轻聊一次，跟进状态、收集信息

**面试课程密集期**：随时关心学习压力、作业进度、心态变化

**夏校 / 关键活动前**：专门沟通一次，梳理机会、提醒重点、同步核心老师

**材料复审 \& 申请递交前**：安抚焦虑、打气鼓励、全程陪伴

四、最终效果

学生不会觉得自己只是 “被上课、被交付服务”

全程有**亲近的学姐陪着、懂他、支持他、帮他搭桥**

所有机会、信息、困惑都能被及时接住，不会错过任何优势

师生关系极强、信任感拉满、情绪稳定、申请状态更好

整个服务从 “课程教学” 升级成**真诚陪伴式高端升学支持**
