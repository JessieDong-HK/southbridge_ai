# 10 · 学生画像库

> 用途：站在学生视角，还原不同背景学生的完整决策路径和痛点。用来持续验证和修正标签系统——每次新增画像，检查能力标签是否有缺口。

---

## 画像 1｜陆本转美本学生

> 来源：Jess 口述（2026-06-18）

### 基本画像

在中国普通大学（非中外合办）念书，想转到美国本科。

### 完整痛点路径

**阶段 1｜我连这是怎么回事都不知道**
- 这条路非常小众，网上信息极少
- "我家出得起学费，但具体怎么转？"
- 想知道：时间线、要准备什么、要交哪些材料、整个流程是什么样的
- → 需要有人先告诉我"这到底是怎么回事"

**阶段 2｜我能去哪里**
- 美国所有学校都接受我这样背景的学生吗？
- 我应该申请哪些学校？概率大不大？
- 选什么专业？
- 不同方向（艺术/理科/工科/商科/文科）的学生策略不同——比如艺术生想转电影，但很多学校没有电影专业，可能要转 media/communication
- → 需要有人帮我做选校定位 + 方向规划

**阶段 3｜哪个学校适合我**
- 知道了哪些学校够得到，但哪个适合我？
- 在意的点：学校 title、未来能提供什么发展资源、去了到底能获得什么
- 说实话，我自己都想不明白未来要怎么发展
- → 需要有人帮我深度分析 fit + 未来发展可能性，听案例、听分析

**阶段 4｜申请材料怎么搞（中国大学特有）**
- 中国大学和美国大学的材料体制不同
- 成绩单要翻译、GPA 要转 4.0 体制、要公证
- 推荐信：中国老师不亲自写——需要自己以教授口吻起草，突出申请人设，教授过目签字
- 还要密封、盖学校章、寄出去
- → 需要一个走过这条路的人告诉我具体每一步怎么操作

**阶段 5｜不同学校偏好不同，文书怎么打**
- 有的学校喜欢学术强的，有的喜欢多元发展的，有的喜欢 work hard play hard vibe
- 文书里该写什么来体现我的特点？
- → 需要有人帮我做学校偏好的深度分析 + 文书方向指导

**阶段 6｜文书支持（不同深度）**
- 情况 A：我能力很强，大概知道怎么写，只需要有人指点方向 + 帮我看 + 来回改几轮（给 feedback → 我改 → 再给 feedback → 我再改）
- 情况 B：我是普通学生，需要有人告诉我文书框架和结构怎么搭
- 情况 C：需要有人直接帮我写/改写
- 情况 D：需要有人帮我做 brainstorming 挖掘经历

**阶段 7｜长期陪伴**
- 转学是至少好几个月的事，会很需要有人陪着走过
- 情绪支持本身就是最好的支持——有个人一直在那里
- 有的人只需要 1-2 次咨询就解决了，有的需要长期陪伴

### 标签覆盖检查

| 痛点 | 对应标签 | 是否覆盖 |
|------|---------|---------|
| 基本信息获取（这是怎么回事） | ❓ 该不该走这条路 | ✅ |
| 时间线 & 准备 | 📊 完整 school list, 🧠 长期陪伴 | ✅ |
| 哪些学校接受我 | 🏫 选校方向建议 | ✅ |
| 方向策略差异 | 🎨 活动规划 & 人设 + mentor 自身背景 | ✅ |
| 深度 fit 分析 + 未来发展 | 🔍 深度 fit 分析 | ✅ |
| 材料准备（翻译/公证/密封） | 📋 材料准备 & 流程 | ✅ 本次新增 |
| 推荐信撰写 | ✍️ 推荐信撰写 | ✅ 本次新增 |
| 学校偏好 → 文书方向 | 🔍 深度 fit 分析 + 💡 brainstorming | ✅ |
| 文书：指点 + 来回改 | 👀 给 feedback + ✍️ 深度修改 polishing | ✅ |
| 文书：框架搭建 | 🏗️ 文书框架搭建 | ✅ |
| 文书：直接写 | 🖊️ 直接写/改写 | ✅ |
| 长期陪伴 + 情绪 | 🧠 长期陪伴 accountability | ✅ |

> 结论：画像 1 的痛点基本被现有 23 个标签覆盖。新增了两个标签（✍️推荐信撰写、📋材料准备&流程）。
