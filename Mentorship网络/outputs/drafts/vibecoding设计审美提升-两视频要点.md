> 来源：小红书 @HiHi小壮 两个视频
> 用途：提升 Mentorship 小程序 UI 设计质量

---

## 视频 1：三个审美库 + 两个技巧

### 三个设计参考网站

| 网站 | 定位 | 怎么用 |
|------|------|--------|
| **Pageflows** | 收录全球一线产品的完整 UI 页面和用户流程录像 | 看顶尖公司怎么做交互——Lovable、Cloud 的真实页面流程 |
| **Godly** | 偏高级感、创意感、大胆前卫的视觉交互 | 适合艺术/人文风格的产品。看到喜欢的交互录屏给 vibecoding 工具模仿 |
| **Mobbin** | 按行业和 UI 元素筛选，细节层面案例 | 某个页面不会做（搜索页/设置页/弹窗）→ 直接来这里对着抄。兼具创意和落地性 |

### 两个技巧

**技巧 1：截图直接复刻**
- 看到喜欢的 UI → 截图/录屏 → 丢给 vibecoding 工具
- Prompt：`参考附件中的UI设计风格，帮我统一优化UI视觉，功能保持不变`

**技巧 2：让 AI 帮你提炼风格**
- UI 截图 → 丢给 Gemini → 让它总结：
  - ① UI 风格关键词
  - ② 交互逻辑
  - ③ 配色方案
- 再把 AI 提炼的关键词喂给 vibecoding 工具

## 视频 2：三招做出高级感

### 第 1 招：1:1 精准复刻
- 录屏好看的网页 → 喂给 Kimi 2.6
- Prompt：`请复刻附件视频中的视觉网站UI设计风格和配色元素保持一致1比1完整还原`
- 先模仿，再创作

### 第 2 招：学会三个前端术语

| 术语 | 含义 | 对我们 UI 的影响 |
|------|------|---------------|
| **Hero 区** | 网页首屏——用户第一眼看到的区域 | 首页 Hero 可以放一段符合调性的全屏视觉 |
| **Shader** | 动态效果——流动的光、扭曲的粒子、迷幻背景 | 可以做微妙的背景动态，不抢内容但有质感 |
| **3D / JSON** | 3D 视觉效果，立体感和空间感 | 卡片可以有微妙的纵深，不是纯平面 |

### 第 3 招：善用高质量模板
- Kimi 2.6 提供动态模板
- 框架拿来 → 替换成自己的内容（图片、视频、文案）

## 我们可以立刻做的事

1. **去 Mobbin 搜** mentorship / education / community 类产品的 UI 案例，找到对标 → 截图 → 分析设计模式
2. **去 Pageflows 看** 完整用户流程——特别是 mentor 浏览 → 筛选 → 预约这个流程，参考成熟产品怎么做
3. **去 Godly 找** 温暖、人文气质的交互参考——看看暖色调社区产品怎么做动画和交互
4. **把我们 v2 原型的截图丢给 Gemini**，让它提炼风格关键词，看跟我们品牌规范是否一致
