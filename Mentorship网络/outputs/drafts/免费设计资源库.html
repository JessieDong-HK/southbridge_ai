01
图标库（替代系统 Emoji）
系统 emoji 在不同平台显示效果不一致，而且风格太"廉价"。以下开源图标库都是 SVG 矢量格式，统一风格，可以自由改色。
★
Phosphor Icons ⭐ 首选
phosphoricons.com
目前最全面的开源图标库——1,488 个图标，6 种风格（thin/light/regular/bold/fill/duotone）。 风格干净现代，圆角线条。有 Figma 插件可直接拖拽。支持 React/Vue 组件。
首选推荐 MIT 开源 Figma 插件 SVG · React · Vue
2
Pikaicons
pikaicons.com
专为"温暖友好"风格设计。圆角柔和，有"精品感"。数量不如 Phosphor 多但风格更独特。
温暖风格 MIT Figma
3
Lucide Icons
lucide.dev
1,000+ 图标，Feather Icons 的社区 fork。风格干净简洁，Figma 插件可用。
ISC（≈MIT） Figma 插件
4
Iconoir
iconoir.com
1,597 个图标，MIT 协议。量最大，风格偏中性现代。line + fill 双风格。
MIT Figma
5
MingCute Icon
mingcute.com
国内设计师出品，2,400+ 图标。有填充和线框两种风格。对中文场景友好。
MIT SVG · Figma
💡 建议
主力用 Phosphor Icons（量大、6 种风格、Figma 插件、MIT）。
需要"更温暖"的图标时，从 Pikaicons 补充（风格更软但量少）。
能力标签和赛道卡片 保留少量 emoji 作为品牌记忆点，但用自定义 SVG icon 替换大部分系统 emoji。
02
人物插画（替代照片 / 装饰用）
当 mentor 没有照片、或需要装饰性配图时使用。全部免费可商用。
★
Open Peeps ⭐ 强烈推荐
openpeeps.com
手绘涂鸦风格的人物插画库。可以自由混搭——换发型、表情、服装、配饰——组合出几十万种不重样的人物。 这可能是我们 mentor 头像的最佳替代方案：没有照片的 mentor 用 Open Peeps 生成一个代表自己的卡通形象，比系统 emoji 有个性得多。
强烈推荐 CC0（完全自由） SVG · PNG · Figma
2
unDraw
undraw.co
1,306+ 幅扁平插画。内置调色器，一键换成我们的暖珊瑚色。MIT 协议，Figma 插件。
MIT Figma
3
Blush
blush.design
多个艺术家合集，各种风格可选。Figma/Sketch 插件可用。部分免费（CC0/免费包），部分付费。
部分 CC0 Figma
4
illlustrations
illlustrations.co
120+ 幅多彩温暖插画，场景丰富。CC0 协议完全自由。有 Figma 插件（1000+ 幅）。
CC0 Figma 插件
5
OpenStickers
openstickers.craftwork.design
80 幅阳光可爱贴纸风插画，涂鸦质感温馨。适合做点缀元素、活人标签旁边的装饰。
免费商用 SVG · Figma
03
背景纹理 & 图案
纯色底太素，照片太重。加一点淡淡的纹理或图案让页面有"质感"而不抢内容。
1
Hero Patterns
heropatterns.com
可重复 SVG 背景图案。自定义颜色和透明度。选暖珊瑚色调 → 下载 SVG/CSS。
免费 SVG · CSS
2
Pattern Monster
pattern.monster
在线图案生成器，滑块调参——形状、大小、间距、颜色。导出 SVG/CSS。
免费 SVG · CSS
3
Transparent Textures
transparenttextures.com
纸张、布纹、水彩等自然纹理。叠加在奶油白底色上，增加"纸质"的温度感。
免费 PNG · CSS
4
CSS Background Patterns
codepen.io (Temani Afif)
纯 CSS 写的图案——零图片加载。适合做极简几何纹理。完全自定义颜色。
开源 纯 CSS
💡 用法建议
在页面背景（奶油白 #FFF8F0）上叠加 8-15% 透明度的暖色调几何图案，或者叠加 Transparent Textures 的纸张纹理。 效果：页面看起来像一本温暖的书，而不是冷冰冰的白色 App。不抢内容，但感受完全不同。
04
总结：推荐组合
图标：主力 Phosphor Icons（量大、6 风格、Figma、MIT）+ 少量 Pikaicons 补温暖风格
人物头像（无照片时）：Open Peeps——每个人可以捏一个代表自己的卡通形象
装饰插画：unDraw——用内置调色器一键换珊瑚色
页面背景：Hero Patterns 或 Transparent Textures——叠加在奶油白底色上
点缀元素：OpenStickers——小贴纸风，放活人标签旁边
