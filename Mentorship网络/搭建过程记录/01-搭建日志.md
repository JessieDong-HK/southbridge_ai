## 2026-06-18 · Mentor 池整合 + 标签系统三轮迭代

**做了什么**：
- 合并两批 mentor 数据：9 人信息表 + 6 人转学访谈 → 15 人完整池
- 标签系统从 v1 到 v3 的完整演化：
  - **v1（废弃）**：60+ 二级标签，服务视角，嵌套在赛道下
  - **v2（废弃）**：改为学生视角三层（赛道→痛点→风格），但痛点太模糊
  - **v3（当前）**：赛道和能力从嵌套改为**平行双维度**——赛道 ∥ 能力，能力跨赛道通用
- 新标签结构：维度一 赛道（6选1）+ 维度二 能力（7域23标签）+ 维度三 活人标签 + 维度四 可约服务
- 根据 Jess 的「陆本转美本」学生画像，补充了两个能力标签（推荐信撰写、材料准备&流程）

**关键决策**：
- 赛道和能力不是嵌套关系——mentor 的文书能力不因帮转学生还是申美本的学生而变
- 活人标签不做分类，唯一目的是让人觉得 mentor 是活人
- 分享会嘉宾不在个人 profile 展示

**产出文件**：
- mentor访谈/02-Mentor初选池-信息汇总.md

## 2026-06-18 · 竞品深度调研（5 份报告）

**做了什么**：
- ADPList 深度 UX 拆解（页面结构、标签体系、预约流程、入驻机制、已知问题）
- 9 家 mentorship 平台商业模式对比（6 种盈利模式）

- 竞品页面结构对比（ADPList/MentorCruise/Leland 的导航、卡片、预约、Dashboard 设计）

**关键发现**：
- 四家起步全部是供给端先行——先攒 mentor，需求跟着 mentor 的 network 来
- MentorCruise $0 融资，5 人团队，$506K 年收入，$500K 利润
- 没有直接竞品——所有现有平台都是职业发展赛道，教育 mentorship + community-driven 没人做
- 付费 mentor 市场在涨（UK +340%），免费在降（-23%）

## 2026-06-18 · 运营流程 + 品牌识别 + UI 原型 v2

**做了什么**：
- 设计完整平台运营流程：学生多选时间→mentor 择一确认→自动建会→提醒→反馈，含 mentor 不回复升级机制
- 确定品牌方向 A（温暖人文·珊瑚色系）
- 定义完整品牌识别系统：色彩、字体、圆角、间距、图标（Phosphor Icons）、插画（Open Peeps/unDraw）、背景纹理、文案语气
- 收集免费开源设计资源（Phosphor Icons、Open Peeps、unDraw、Hero Patterns 等）
- 用品牌规范重做小程序 UI 原型 v2（8 个屏幕，手机宽度渲染）
- 将功能 emoji 替换为 Phosphor SVG 图标

**关键决策**：
- 微信小程序为主，网站为辅
- MVP 先做展示+下单+支付+半自动通知，后续接入腾讯会议 API
- 品牌调性：温暖、可信赖、有烟火气，不用营销腔
- 图标：功能 icon 用 Phosphor，赛道卡片和能力标签保留 emoji 作为品牌记忆点

- 安装 AI Website Cloner Template，用 Playwright + AI 逆向工程 [follow.art](https://follow.art/) 首页

  - **Hero** — 浮动图标 + 视差滚动 + 淡入
  - **How It Works** — IntersectionObserver 滚动触发入场
  - **Testimonials** — 自动轮播 (5s) + 手动切换
  - **Centralize** — 3D 翻转卡片 (hover 触发)
  - **Card Showcase** — 鼠标跟随 3D 透视效果
  - **Audience Support** — 无限循环滚动轮播 + hover 暂停
  - **Connectory** — 滚动入场交互动效
  - **Join Us + Footer** — CTA hover 缩放
- 整理到 `参考素材/克隆网站/follow-art/`，含完整 README_CLONE.md

**关键收获**：
- Follow.art 的设计语言对 mentorship 网站有参考价值：超大标题 (max 257px)、品牌色分块蒙层、sticky scroll layers、直接赞助功能
