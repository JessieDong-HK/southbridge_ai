# 01 · 竞品调研

> 来源：市场调研.docx + Jess 与 Alicia 的 AI 留学平台对标交流（2026-06-02）

---

## 一、Degree Mentors

- 网址：https://www.degreementors.org/services#pricing
- **运营状况**：一般——博客仅更新一篇，产品介绍非常简略
- **核心模式**：先提供一次免费咨询通话 → 了解背景和申请目标 → 出具时间线和规划方案（含文书、活动规划数量与方向、选校等）
- **套餐设计**：非常简单，本科仅分 UC 和 Common Application 两个套餐
- **定位判断**：与其说是 mentor，不如说更像中介（类似国内小型工作室）
- **可参考的服务内容**：
  - 个人洞察问题（PIQ）头脑风暴
  - 活动列表优化
  - 选校与选专业策略制定
  - 申请材料审核
  - 最终提交材料校对（proofread）

## 二、Your Admissions Mentor

- 网址：https://youradmissionsmentor.com/
- **整体评价**：做得不错，真诚度和可信度明显更高
- **运营者**：Maddie 个人运营的工作室
- **优点**：
  - "Learn About Me"板块出彩：完整写了个人 journey、values、教学与文书经验
  - 展示学生反馈：聊天记录、录取报喜截图、学生评价
  - 个人 IP 感强：用真实故事和成果建立信任 → **完全可以借鉴**
- **明显缺点**：
  - 价格与套餐不透明：完全不公开，只能私聊咨询，体验差
- **服务内容**（本质偏中介）：
  - 全程导师辅导（mentorship）
  - 文书故事线构建（essay narrative）
  - 活动列表与简历优化（activity and resume）
  - 面试策略指导（interview strategy）

## 三、College Advisor

- 网址：https://www.collegeadvisor.com/pricing/
- **定位**：完完全全的中介机构，并非 mentor 模式

## 四、ADP List

- **整体评价**：布局远比预想中简洁，体量庞大
- **模式**：自由入驻——师生可直接入驻发布简介接单
- **问题**：
  - 缺少严格审核与内容优化指导
  - 导师个人简介大多撰写随意潦草
  - 信息量甚至少于领英（不标注任职企业，经历描述笼统模糊）
  - 单次预约 ~$100+，但用户无法直观了解导师详细实战经历
  - 用户评价缺乏实质性内容
- **启发**：
  - 平台无需做得精致花哨也能正常运营
  - 行业实操难度比预想更低
  - **推测其初期策略**：先精心打造 ~50 名核心导师 → 依靠优质初始导师打响口碑 → 后期开放自由入驻
- **标签分类模式**：导师注册时勾选职业/学历/从业经历等标签 → 方便用户筛选，但分类偏笼统

## 五、纯公益平台（名称未记录）

- **定位**：纯公益的国际化精英教育链接平台
- **目标**：帮助申请藤校/Top30 的学生链接到真实在校生
- **当前状态**：不收取任何费用，已有 6 名导师确认出席
- **背景**：有官方机构（非留学申请机构）资助，试运行阶段完成后对外公开
- **门槛**：对学员有一定学术背景要求，符合标准后跟进面试

## 七、竞品参考：AI 留学平台（Alicia 的项目）

来源：2026-06-02 Jess + Alicia 交流

- **方向**：AI 驱动的留学咨询 SaaS（类似 mentorship marketplace）
- **市场**：澳洲留学，20+ 导师
- **当前状态**：平台已能用但还在内测，"PMF 还没跑通"
- **真实数据**：
  - 做了 40+ 用户访谈 → 发现澳洲留学生付费意愿不强
  - 小红书只来了 1 个真实客户
  - 留学市场萎缩 + AI 工具兴起 → 轻咨询需求在下降
- **相关信息**：
  - 竞品"寻师"已断更
  - "学迹APP"态度封闭不愿合作

## 关键洞察总结

| 洞察 | 对我们的启示 |
|------|-------------|
| 竞品大多偏中介，真正 mentorship 定位的少 | 差异化空间大 |
| ADP List 初期靠精选导师起量 | 我们首批 20 位 mentor 深度打造策略正确 |
| 个人 IP + 真实故事是信任核心 | Maddie（Your Admissions Mentor）模式值得借鉴 |
| 价格透明是用户痛点 | 我们的价格体系应该公开 |
| 澳洲市场付费意愿低 | 英美转学市场可能更健康 |
| AI 工具兴起可能影响轻咨询需求 | 需要持续关注 PMF |
