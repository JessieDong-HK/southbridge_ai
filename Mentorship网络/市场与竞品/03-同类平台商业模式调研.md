# 03 · 同类 Mentor 平台商业模式调研

> 调研日期：2026-06-18。覆盖 9 家 mentorship/coaching 平台，聚焦商业模式、定价策略、mentor 入驻机制。

## 二、六种商业模式

### 模式 1｜交易抽佣（Leland）

- 教练自定价，平台抽 **15-30%**
- Leland 2024 年拿了 $12M Series A，4x 年增长，100,000+ 用户，现金流为正
- 核心壁垒：**5% 教练接受率** → 高质量供给 → 口碑传播 → 75% 有机增长
- 给教练提供"business-in-a-box"：支付、排期、CRM、AI session 总结
- 同时有免费直播、录播课、社区做流量入口

> **对我们的启示**：交易抽佣是最自然的后端变现方式。Leland 证明了「高门槛 → 高质量 → 高信任 → 高转化」这个闭环。

### 模式 2｜会员订阅 + 无限 session（GrowthMentor）

- $49-99/月 → **无限次 mentor 通话**（但最多同时约 3 个）
- Mentor 由平台付费，不向 mentee 额外收费
- ~85% 的 mentor 接受平台定价；少数可额外收费（鼓励"startup-friendly"定价）
- 新 mentor 先做免费 session 攒 reviews → 再进入付费池

> **对我们的启示**：无限 session 模型适合"需要频繁找不同专家"的场景。对我们不太适用——教育 mentorship 是深度陪伴，不是高频碎片化咨询。

### 模式 3｜导师自定价 + 按月订阅（MentorCruise）

- 每个 mentor 设自己的月费（$50-400），通常含 2-4 次 call + 无限异步沟通
- 也有一次性 session：Intro Call ($39)、Work Review ($89)、Interview Prep ($99)
- 7 天免费试用，无锁定期
- **核心差异**：深度长期关系 vs 广度碎片化咨询

> **对我们的启示**：跟我们最像。我们的产品 3（长期 mentorship）就是这种模式——mentor 按月/周期收费，长期陪伴。区别是我们的 mentor 不自己定价，我们有四档产品价格。

### 模式 4｜按分钟付费（Clarity.fm）

- ~$1.60/分钟，一次 30 分钟约 $50
- 适合：一个具体问题，需要快速答案
- 无持续性，无长期关系

> **对我们的启示**：这就是我们的产品 1（¥599 分享会 + 30min 1v1）的定价逻辑基础。短咨询 = 解决一个具体问题，不需要长期绑。

### 模式 5｜积分/信用订阅（MentorPass）

- $300-1,200/月，消耗积分约 session
- 400+ 行业领袖级 mentor
- 高端定位，价格筛选用户

### 模式 6｜企业 B2B（BetterUp, MicroMentor）

- BetterUp：~$250/人/月，只卖企业不卖个人
- MicroMentor：非营利，企业和基金会赞助 → 免费提供给创业者
- 企业合同比 C 端消费更高利润、更稳定

> **对我们的启示**：B2B 是远期方向——卖给留学机构、国际学校、教育集团，他们给自己的学生配 mentorship 福利。

## 三、关键行业趋势

### 付费在涨，免费在降

| 指标 | 变化 |
|------|------|
| UK 付费 mentoring | **+340%** |
| UK 免费 mentoring | **-23%** |
| UK  mentoring 市场总额 | £182M（2024），2019 年仅 £67M |
| 在线 mentoring | **89%** 已转为线上 |

> 学生愿意为真实的过来人经验付费——免费模式反而在萎缩。

### 质量策展是核心壁垒

| 平台 | 接受率 | 效果 |
|------|--------|------|
| Leland | ~5% | 75% 有机增长，现金流为正 |
| GrowthMentor | ~5% | 高信任，高转化 |
| MentorCruise | <5% | 每个 mentor 都经过审核 |
| ADPList | ~30% | Trustpilot 2.6/5，no-show 泛滥 |

> 宽松入驻 → mentor 质量参差 → 学生体验差 → 评分低。**高门槛是护城河。**

### "Business in a Box" 是留住 mentor 的关键

Leland 给教练提供：支付、排期、CRM、消息、时间追踪、优惠码、AI session 总结。MentorCruise 有进度追踪、目标框架、session 笔记库。

> 我们 MVP 阶段先手动，但长期要给 mentor 提供工具——至少包含：日历同步、自动提醒、session 记录、收入追踪。

### AI 正在渗透

- Leland：AI onboarding → goal → 匹配 coach；AI 生成 coach 资料
- GrowthMentor：AI 匹配引擎（描述痛点 → 推荐 3 个 mentor）+ AI session 总结
- Get Mentors：450+ AI mentor 角色，$9.99/月

> AI 当前在"匹配"和"session 总结"两个环节最有价值。对我们来说，AI matching（根据学生画像 + 需求匹配 mentor）是中期可以加的功能。

## 四、对我们的定位判断

| 维度 | 我们 | 对标谁 |
|------|------|--------|
| 定价模式 | 四级产品（¥99 → ¥30K） | MentorCruise 的分层 + Leland 的交易抽佣 |
| 供给质量 | Jess 二次确认（手动策展） | Leland / MentorCruise 的高门槛 |
| 支付方式 | 按次 + 按周期 | MentorCruise（月付）+ Clarity（按次） |
| 赛道 | 国际教育（非职业发展） | 独特——目前没有直接竞品 |
| 社区感 | 半公益 community-driven | 跟所有纯交易平台都不一样 |

**核心差异化**：
- 不是纯交易平台（Leland）——我们是 community，mentor 留在这里不是因为钱
- 不是纯职业平台（ADPList / GrowthMentor）——我们是教育赛道
- 不是纯免费（ADPList / SCORE）——我们有清晰的付费产品线
- 不是纯一对一（MentorCruise）——我们有多 mentor 协同的长线套餐（待设计）
