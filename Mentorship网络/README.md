# 国际教育 Mentorship 网络

## 这是什么？

一个多切入点的国际教育 peer mentorship 平台——连接有真实经历的名校学长学姐和迷茫的学生。

**第一个切入点：美本转学。** 后续扩展：美本申请、海外高中、高考→留学、陆本→美本等更多赛道。

**三种读者，一个知识源：**
- 🧑‍💼 **团队成员 / Mentor** — 了解产品体系、社区文化、参与方式
- 🤖 **AI Agent** — 内容创作、客服回复、运营的上下文注入
- 👤 **管理者（Jess）** — 项目全貌、进度追踪、决策依据

## 如何使用

1. **新人了解项目**：先读 [项目总览.md](项目总览.md)
2. **产品设计参考**：读 [产品与运营/01-产品体系设计.md](产品与运营/01-产品体系设计.md)
3. **Mentor 招募与社区**：读 [产品与运营/02-Mentor社区与文化.md](产品与运营/02-Mentor社区与文化.md)
4. **当前推进**：读 [产品与运营/04-当前推进策略.md](产品与运营/04-当前推进策略.md)
5. **竞品与市场**：读 [市场与竞品/01-竞品调研.md](市场与竞品/01-竞品调研.md)
