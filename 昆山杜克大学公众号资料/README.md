# 昆山杜克大学 · 公众号资料库

> 数据来源：DKU 官方微信公众号 780 篇帖子（2017年9月 — 2026年6月）
>
> 用途：补充官网资料库，提供更丰富的学生叙事、招生实操指南、校园体验、毕业生发展数据

## 与官网资料库的关系

| 维度 | 官网资料库 `昆杜官网资料/` | 公众号资料库 `昆山杜克大学公众号资料/` |
|------|--------------------------|---------------------------------------|
| 数据源 | 680页官网JSON爬虫 | 780篇公众号推文txt |
| 内容特点 | 结构化、官方、覆盖面广 | 叙事化、口语化、深度更强 |
| 学生故事 | 8篇英文叙事（简历式） | 100+篇中文深度访谈（完整成长叙事） |
| 招生信息 | 录取要求概述 | ⭐ 详细申请指南（文书/推荐信/AI面试/系统操作） |
| 教育理念 | 7项原则、官方使命 | ⭐ 招办主任深度阐述、理想学生特质 |
| 校园生活 | 设施介绍、社团列表 | ⭐ 第一人称就读体验、新生适应故事 |
| 毕业生数据 | 基础统计表格 | ⭐ 四届完整报告、博士录取全景、校友追踪 |
| 官方金句 | 校长/教授英文原话 | ⭐ 中文语境金句、毕业演讲全文 |

> **核心原则**：公众号资料库不重复官网已有的内容，而是补充官网没有的深度、细节和叙事。

## 与其他知识库的关系

```
昆杜官网资料/          ← 官方结构化事实（8个文件）
昆山杜克大学公众号资料/  ← 公众号叙事补充（本文件夹，6个文件）
    ↓ 两者共同支撑 ↓
上纽昆杜升学辅导/      ← 业务知识库（面试/文书/课程中使用这两套资料）
```

- 官网资料库提供"骨架"（事实、数据、结构）
- 公众号资料库提供"血肉"（故事、体验、细节）
- 升学辅导知识库是"使用者"——面试模拟、文书指导、家长沟通时按需调用
