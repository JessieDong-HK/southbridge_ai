# 📋 更新总管理 Skill

一句话刷新 Jess 全部项目的最新进展到 `总管理.md`。

---

## 执行步骤

### Step 1：读取所有进度源文件

按顺序读取以下文件，提取每个项目的最新条目（只看日期标题 + 做了什么 + 下一步）：

| 项目 | 进度源文件 |
|------|-----------|
| 上纽 · 咨询管理 | `上纽昆杜升学项目/咨询管理平台/搭建记录/01-搭建日志.md` |
| 上纽 · AI面试 | `上纽昆杜升学项目/AI面试平台/搭建记录/01-搭建日志.md` |
| 上纽 · 自媒体 | `上纽昆杜升学项目/自媒体内容产出/搭建记录/01-搭建日志.md` |
| Mentorship网络 | `Mentorship网络/搭建过程记录/01-搭建日志.md` |
| 个人IP | `Jess个人IP/搭建过程记录/01-搭建日志.md` |
| 人员管理 | `人员清单.md` |
| 创业营 | ⚠️ 暂无源文件，标记为「待建」 |
| AI生物医药 | ⚠️ 暂无源文件，标记为「待建」 |

读取方式：每个文件只需读前 50 行即可覆盖最新条目。用 `grep "^## 2026"` 快速定位日期条目。

### Step 2：提取摘要

对每个项目，提取：
- **最近进展**（1-2 句话，来自最新 2-3 条日志的标题）
- **下一步**（来自最新条目的「下一步」字段）
- **状态**：✅ 完成 / 🔧 迭代中 / 📋 待开始 / 🚀 即将启动

### Step 3：刷新总管理.md

更新 `总管理.md` 的「项目总览」表格中的：状态、最近进展、下一步。
更新「最后更新」日期为今天。

### Step 4：不要做的事

- ❌ 不要添加项目细节（想看细节点进度源文件链接）
- ❌ 不要添加会议索引（会议记录在 `会议记录/` 独立管理）
- ❌ 不要改动表格外的其他内容
- ❌ 不要修改各项目的搭建日志（那是各项目自己的事）
- ❌ 不要为没有源文件的项目编造内容
