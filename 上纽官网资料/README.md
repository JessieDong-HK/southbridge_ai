## 数据质量说明

上纽爬虫质量显著优于昆杜：
- **50.6%** 的页面超过 200 字（昆杜仅 ~35%）
- 学生故事中位字数 **981**（昆杜仅 ~500）
- 有 **166 篇**互动专题（is/）+ **18 篇**学生博客 — 昆杜没有这些内容类型
- 中文内容较充足（750页），但部分中文页面比英文薄

**已知缺口**：
- 学费具体数字（爬虫未获取到）
- 详细招生要求和截止日期（招生板块仅9页，内容较薄）
- 部分领导团队成员详细个人简介缺失
- 毕业生精确统计数据（以PDF附图形式存在，爬虫未能提取）
