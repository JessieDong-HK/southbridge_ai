# 学术写作辅导

## 所属阶段

复审（校园日）阶段 · 两校均有（上纽校园日有单独限时写作）

## 服务目标

帮助学生输出结构清晰、论证严谨、表达准确的学术文章，达到两校的学术写作标准。

## 服务内容（2课时）

### 1. 结构训练
- **Introduction** — 如何写清晰有力的开篇：Background → Problem/Thesis → Roadmap
- **Body Paragraphs** — 主题句 → 证据/论证 → 解释 → 回到论点的闭环
- **Conclusion** — 总结而不重复，展望而不啰嗦

### 2. 论证能力
- 如何用一个具体例子支撑一个抽象论点
- 如何回应和平衡反方观点（Counterargument）
- 常见的逻辑谬误识别和避免

### 3. 表达提升
- 学术英语的核心特征：正式但不啰嗦、准确但不生硬
- 告别"中式英语"：常见表达问题的系统性纠正
- 连接词和过渡的精准使用

### 4. 限时写作训练
- 模拟校园日限时写作场景
- 培养"拿到题目 → 快速构思 → 高效成文"的能力
- 时间管理技巧：多少分钟做计划、多少分钟写、多少分钟检查

## 核心理念

> 学术写作不是展示词汇量，而是展示思维的清晰度和论证的严谨度。

## 与文书辅导的区别

| | 文书辅导 | 学术写作 |
|------|------|------|
| 场景 | 初审申请文书 | 校园日限时写作 / 学术作业 |
| 目的 | 展示个人特质和匹配度 | 展示思维和学术能力 |
| 文体 | Personal Statement / Essay | Academic Writing |
| 形式 | 提前准备、反复打磨 | 限时写作、即时输出 |
| 个人性 | 强（讲自己的故事） | 弱（分析客观议题） |
