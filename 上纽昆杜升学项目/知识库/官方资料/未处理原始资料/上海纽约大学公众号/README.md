## 二、内容分类

根据标题和内容，997篇帖子可归为以下主要类别（一篇帖子可能跨类）：

### 🔥 A类：学生人物故事（~119篇）

**这是公众号最有价值的内容类型。**

每年毕业季（3-6月）集中发布毕业生人物特写，每篇 3,000-8,000 字，包含完整叙事弧线：

- **毕业生人物系列**：每年约 15-25 篇，如"上纽大2026届毕业生Raquel Jimenez"、"蹦床·国关·体育政治｜AJ Noble的上纽闭环"
- **校友追踪**：如"上纽大首届校友吴雨蔚：快消大厂市场人养成记"
- **新生故事**：如"2022级新生故事｜我与上纽的双向奔赴"
- **博士生风采**：如"上纽大首位神经科学博士毕业生——朱小悦"
- **特殊荣誉人物**：苏世民学者、燕京学者、富布赖特奖、伊拉斯谟奖学金获得者

**特点**：
- 全部为中文叙事，比官网英文故事（中位981词）更深、更长、更有画面感
- 包含大量可直接引用的原话、场景描写、情感转折
- 覆盖背景极其多元（哥斯达黎加、亚美尼亚、阿肯色小镇、西雅图华裔、收养家庭……）

### 🔥 B类：毕业典礼演讲（~25篇/届 × 5届）

每一年毕业典礼都有 **6-8 篇**独立帖子，分别发布：

| 演讲者 | 内容 |
|--------|------|
| 荣誉演讲嘉宾 | 杨扬(2022)、崔天凯(2023)、姚明(2024)、孔祥东(2025)、陈冲(2026) |
| 校长 | 童世骏（2022-2026每年） |
| 常务副校长 | 雷蒙（2022-2026每年） |
| 教务长 | 衞周安(2022-2025)、吴蓓(2026起) |
| 纽约大学校长 | Andrew Hamilton(2022-2023)、Linda G. Mills(2024-2026) |
| 中国学生代表 | 施怡聆(2022)、孙蕴喆(2023)、李语童(2024)、等 |
| 国际学生代表 | Bongani Musikavanhu(2022)、全中文发言的国际生(2023)等 |

**特点**：
- 中英双语全文
- 金句密度极高
- 陈冲2026年致辞堪称文学级演讲稿（引用惠特曼、里尔克、厄德里奇）

### A类：教授与研究（~267篇）

覆盖面最广的类别，包含：

- **教授人物特写**：如"吴晓刚：做经世致用之学问"、"光影五十年：衞周安与中国的半世纪情缘"
- **研究突破报道**：如"上纽大首席科学家成功构建首个视网膜拓扑投射图数学模型"
- **新教师介绍**："师资队伍再壮大！二十名新教师加入上海纽约大学"
- **学术活动报道**：研讨会、论坛、峰会
- **奖项与荣誉**：白玉兰纪念奖、宝钢优秀教师奖、国家自然科学基金等

### B类：校园活动与生活（~245篇）

- **期末秀**：每年12月和5月视觉艺术/交互媒体学生作品展
- **TEDx上纽大**：每年一届
- **社团活动**：Green Shanghai、阿卡贝拉合唱团、学生乐队、Reality Show
- **公益实践**：春假服务行、云贵地区公益、崇明岛湿地保护
- **黑客松**：70位极客24小时AI应用开发
- **体育运动**：NYU vs 上海交大篮球赛、中外合作大学杯体育联赛
- **阅读交流会**：《鱼翅与花椒》作者邓洛普等
- **跟着校长看中国**：横沙岛骑行、婺源行、元荡湖骑行

### C类：招生与申请（~140篇）

- **本科招生**：招生简章、城市宣讲、校园日活动、录取通知书发放
- **硕士项目**：NYU Shanghai-NYU Stern 商科硕士每月宣讲会预告
- **博士项目**：每年秋季招生全面启动
- **暑期项目**：中学生/本科生暑期课程、科研体验项目（神经科学/物理学）
- **国际生**：录取学生周、交流生体验

> ⚠️ 约 40% 的此类帖子是 NYU Stern 商科硕士的月度宣讲会通知，对本科升学辅导的直接价值较低。

### C类：就业与毕业去向（~34篇）

- 年度本科毕业生就业质量报告（2021-2025届）
- 年度硕士项目就业质量报告
- 奖学金获得者报道（苏世民、燕京、富布赖特、伊拉斯谟）

### D类：校长/领导层动态（~60篇）

- 校长演讲（开学、毕业、学术论坛）
- 校长媒体专访（"科学的教育意义是让人习惯于讲道理"）
- 领导层获奖（白玉兰奖、中国政府友谊奖）
- 外交与高层来访（布林肯、雷蒙多、伯恩斯等）
- 与纽约大学总部的互动

### D类：其他

- **节日祝福/红包封面**（~30篇）：春节、元宵、中秋、教师节——内容价值低
- **媒体转载**（~18篇）：标记「转」的外部媒体报道
- **项目广告**（~30篇）：NYU Stern硕士、暑期研修、REITs课程等
- **校史/特别策划**（~10篇）：十周年口述历史纪录片、中外合作办学模式探索等长文

## 六、文件命名规则

帖子文件名格式为 `YYYY-MM-DD_标题.txt`：

- `YYYY-MM-DD` 是发布日期
- `标题` 是公众号推文标题（去除特殊字符）

例如：`2026-04-27_慢慢来，比较快 ｜ 上纽大2026届毕业生Raquel Jimenez.txt`

帖子内通常包含：
- 标题（第一行）
- 正文（中文在前，英文在后）
- 推荐阅读（文末互链）
- 图片标注（如"上：2023年暑假返校后……"）

> ⚠️ 帖子中的图片已丢失（仅为文字描述标注），如需原图需从公众号链接重新抓取。
