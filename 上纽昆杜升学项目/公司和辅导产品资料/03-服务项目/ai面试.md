# AI面试辅导

## 所属阶段

初审阶段 · 昆山杜克大学核心环节（上纽不使用）

## 服务目标

通过系统化课程和全真模拟，让学生精准掌握昆杜AI面试的评分逻辑和答题方法，顺利通过初审筛选。

## 什么是昆杜AI面试

昆杜初审阶段，学生需完成一个由AI系统评估的在线视频面试：

- **双机位**要求（正面摄像头 + 侧面监控）
- 每道题有固定准备时间和回答时限
- **二刷规则**：每位学生有两次完成机会
- AI从4个维度自动评分

### AI评分4大核心维度

| 维度 | 考察什么 |
|------|------|
| **内容完整性** | 是否充分回答了问题，有具体细节 |
| **逻辑连贯性** | 结构是否清晰，思路是否连贯 |
| **英语表达** | 流畅度、清晰度（不要求完美口音） |
| **院校匹配度** | 与昆杜教育理念的契合程度 |

## 各课详解（深度版）

### 课前准备：3个故事深度挖掘（60min，与文书课协同）

在AI面试课开始前，文书老师+AI面试老师协同完成三大故事挖掘：
- **软实力故事**（20min）：聚焦领导力、主动性、问题解决、同理心
- **硬实力故事**（20min）：聚焦学科能力、学习力、专业潜质
- **专业方向锁定+专业故事**（20min）：确定专业方向，搭建专业适配故事

### 第1课：STAR框架 + 故事结构化 + 套题 + 追问（90min）

1. 昆杜AI面试核心规则与评分维度解析（10min）
2. STAR框架讲解（5min）
3. 三大故事STAR结构化打磨，现场英文练习纠错（35min）
4. 套题训练：5道高频真题套题实战（15min）
5. 追问模拟：每故事2-4个细节追问点（15min）
6. 课堂总结+Key Takeaway（5min）

**课后作业**：完成三大故事STAR中英文撰写（2.5-3分钟英文版），脱稿练习

### 第2课：作业复盘 + 昆杜关键词 + 专属资源 + 实战（60min）

**课前老师准备**：
- 批改三大故事STAR作业，润色英文表述
- 准备2个与学生专业/故事强绑定的昆杜专属资源（实验室、研究中心、教授项目、RA机会、Study Away校区、特色社团等）
  - 详见 [08-昆杜官方资源库.md](../08-昆杜官方资源库.md) — 六大研究中心、教授名单、研究项目、学生成就案例
- 准备5道高频真题

1. 作业复盘，针对性纠正（10min）
2. 昆杜5大核心关键词+案例讲解+即时抽查（10min）：
   - Liberal Arts、Interdisciplinary、Global Campus & Study Away、Small Class & Mentorship、Community & Social Impact
   - 关键词官方原文和案例详见 [08-昆杜官方资源库.md](../08-昆杜官方资源库.md)
3. 专属资源匹配讲解（15min）— 参考 [08-昆杜官方资源库.md](../08-昆杜官方资源库.md) 中的六大研究中心、教授名单、标志性项目
4. 3道真题问答+逐题纠错（20min）
5. 学生总结+Takeaway（5min）

### 第3课：社会思辨话题（60min）

1. 上节课作业复盘（10min）
2. 思辨题通用高分结构讲解（5min）
3. 高频思辨真题精讲+现场练习（40min）：AI伦理、全球问题、气候变化、教育公平、科技与社会、个人责任与集体
4. 学生总结+Takeaway（5min）

**AI模考双方案**：
- 方案A：自主模考平台（4次模拟机会，自选最佳提交，生成匹配度报告）
- 方案B：1次真人模拟AI面试（完全还原真题、计时、双机位、追问）

### 第4课：模考复盘 + 薄弱项冲刺 + 考前调整（60min）

1. 模考全面复盘（15min）：逐维度分析内容完整性、逻辑连贯性、英语表达、答题节奏、追问应对
2. 个性化薄弱项专项强化（30min）
3. 考前全流程核对+避雷清单（5min）：设备/网络/光线/背景/着装/时限/二刷规则
4. 全程课程总结+考前信心建立（10min）

**考前加油信息**：AI面试前，老师发送专属加油信息

## 常见学生问题

- **"英语口语不好怎么办"** — AI面试不要求完美口音，要求清晰表达 + 有内容。课程中包含英文表述打磨和纠错
- **"不知道会被问到什么"** — 三大核心故事+套题技巧可覆盖95%面试真题
- **"对着摄像头说话很不自然"** — AI模考（平台或真人模拟）让你在正式考核前充分适应
