## Z 同学

| 维度 | 详情 |
|------|------|
| 录取 | **上海纽约大学 金融学+计算机科学** · A档 |
| 奖学金 | **全球典范奖学金** — "对我来说是完全超出预期的惊喜" |
| 背景 | 上海某高中 · 雅思 6.5 · 全包辅导 |
| 特点 | 性格内向，担心在团队活动中无法突出自己 |

### 辅导关键点
- 量身定制全流程方案，鼓励大胆冲刺奖学金
- 重点训练：如何在舒适圈里做好 small talk，展现契合度而非刻意外向
- 校园日前实战模拟 + Go-to Story 完美适配现场题目

### 学生反馈
> "入学后，当天的学生面试官还告诉我，我那天的发言给她留下了极其深刻的印象。"

## M 同学（初审案例）

| 维度 | 详情 |
|------|------|
| 录取 | **上海纽约大学** · A档 |
| 特点 | "每天都在刷题和考试，没什么经历可写，也不知道能写什么" |

### 辅导关键点
- 不增加经历，而是帮她建立一个清晰、统一、有说服力的申请画像
- 基于她的文科背景与兴趣，打造出"具有国际化视野，善于跨文化沟通，对历史学有清晰兴趣"的画像
- Go-to Story：博物馆志愿者经历 + 模拟联合国经历

---

## 关键发现（Cross-case Patterns）

1. **"平庸"的经历可以被重构为"亮点"** — S同学和M同学都觉得"没什么可写"，但南桥能从日常中提炼出与学校匹配的故事
2. **契合度 > 炫目度** — 招生官不是在找"最优秀的人"，而是在找"最适合的人"
3. **真实现场与辅导高度一致** — L同学反馈"考核内容和辅导方向几乎完全一致"，因为南桥导师来自选拔内部
4. **长期关系 > 一次性交易** — 大量学生申请结束后仍保持联系，很多是朋友推荐来的
