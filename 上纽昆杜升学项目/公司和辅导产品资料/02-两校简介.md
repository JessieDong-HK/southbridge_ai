# 02 两校简介

## 为什么是上纽 & 昆杜？

上海纽约大学和昆山杜克大学是中国中外合办大学中**招生体系最独特**的两所。不同于传统高考"一考定终身"，两校采用**综合评价录取模式**，通过多轮筛选考察学生的综合能力。

---

## 招生体系对比

| | 上海纽约大学 (NYU Shanghai) | 昆山杜克大学 (Duke Kunshan) |
|------|------|------|
| 合作方 | NYU + 华东师范大学 | Duke + 武汉大学 + 昆山市政府 |
| 成立时间 | 2012年 | 2013年（2018年首届本科招生） |
| US News 排名 | No.32 | No.7（Duke University） |
| 学位授予 | NYU 学位 + 中国教育部认证 | Duke 学位 + DKU 学位 + 中国教育部认证 |
| 招生方式 | 综合评价录取 | 综合评价录取（541模式） |
| 教学语言 | 全英文 | 全英文 |
| 校园 | 上海浦东 | 昆山阳澄湖科技园（高铁17分钟到上海） |

### 两阶段筛选

```
初审（文书 + AI面试）
    ↓ 
复审 / 校园日（多环节综合考核）
    ↓
录取（A档 / B档 / 入围）
```

### 校园日典型环节

1. **真人面试** — 外籍+中籍双面试官，1v1 或 2v1，全英文
2. **无领导小组讨论** — 5-6人小组，随机抽题，英文讨论+Presentation
3. **学术写作** — 限时写作，考察论证和表达能力
4. **其他互动环节** — 各年有所不同，考察综合素质

## 昆山杜克大学深度介绍

> 信息来源：DKU 官网 680 页爬取数据（2026年6月），官方最新数据

### 学校概况

- 非营利性中美合办大学，杜克大学 + 武汉大学 + 昆山市人民政府合作创办
- 2022年，国家发改委、教育部、科技部联合认定为中外合作大学首个 **"国际合作教育样板区"**
- 校园占地 200 英亩，位于昆山阳澄湖科技园（中国最富裕的县级市）
- 2024年11月，理事会批准扩招至 **2,200 名学生**总容量

|------|------|------|------|------|------|
| 2024年（2028届） | ~10,000 | ~500 | — | 4,705（+41%） | ~5% |
| 2025年（2029届） | **11,884** | **550** | 385 | 165（来自144国） | **~4.6%** |

- 中国申请增长：北京 +40%、上海 +31%（两市均突破千人大关）

### 昆杜的七项教育原则（Animating Principles）

这是昆杜课程设计的核心理念，也是"What DKU looks for in students"的官方答案：

1. **扎根全球主义（Rooted Globalism）**：在不同文化和身份之间自如切换
2. **协作解决问题（Collaborative Problem-Solving）**：综合不同见解解决复杂挑战
3. **研究与实践（Research and Practice）**：将理论联系实践
4. **清晰沟通（Lucid Communication）**：有效沟通，倾听不同观点
5. **独立与创造力（Independence and Creativity）**：自由探究、深度反思
6. **智慧领导力（Wise Leadership）**：具备道德指南针的思考者和实干家
7. **有意义的人生（A Life of Consequence）**：将学术联系到人生大问题

### 昆杜官方定义"理想学生"

> "求知欲强、具有全球视野、有抱负的学生"（Intellectually curious, globally-minded, ambitious students）

核心选拔标准：
- 杰出的学业成就和对学习的热情
- 较强的英语能力
- 对创新性、综合性、跨学科通识博雅课程有浓厚兴趣
- **冒险精神**和全球视野
- **积极主动**，展现为校园和社区做贡献的能力
- **克服困难的能力**（韧性）

### 学位与认证

- 毕业生同时获得：昆山杜克大学毕业证书和学位证书 + **杜克大学学位证书**
- 加入杜克大学超过 **20 万人**的全球校友网络
- Duke 学位获 SACSCOC 认证；DKU 获中国教育部认证

### 学术特色

- 通识博雅教育 + 跨学科框架
- 大二确定专业方向
- **大一即可参与研究**
- 小规模住宿书院制，师生关系紧密
- **Signature Work（标志性项目）**：每位学生完成一年的原创毕业研究
- 有机会在杜克大学达勒姆校区学习

### 奖学金与经济援助

- 基于需求和成绩的奖学金
- 奖学金可覆盖最高全额学费
- 超过 80% 国际学生获得经济援助/奖学金
- 必须在申请阶段申请经济援助（录取后不可补申）

## 与传统高考的本质区别

| 传统高考 | 上纽/昆杜申请 |
|----------|------|
| 只看分数 | 看综合能力：表达、思维、协作、领导力 |
| 一次考试 | 多轮筛选：初审→复审→高考参考 |
| 被动应试 | 主动展示个人特质和匹配度 |
| 标准化考核 | 个性化评估 |

> 这意味着学生需要的不是刷题辅导，而是**思辨表达、小组协作、临场输出的真实能力提升**——这正是南桥的辅导核心。
