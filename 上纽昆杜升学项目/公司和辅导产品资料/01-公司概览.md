## 服务体系总览

### 第一阶段：初审全流程

帮助学生通过上纽/昆杜的文书和AI面试筛选，进入校园日复审。

**通用流程（深度版）**：
```
师生破冰见面会（30min，Cheryl + 核心主讲老师）
    ↓
3个故事深度挖掘（60min，文书 + AI面试双师协同）
    ↓
文书故事线沟通 → 文书定稿沟通 → 材料终审
    ↓
AI面试课程（4节1v1：STAR框架 + 昆杜关键词 + 思辨 + 模考复盘）
    ↓
考前加油 → 正式考核
```

**核心服务**：
- **文书打造**：故事挖掘 → 逻辑构建 → 深度打磨 → 定稿沟通
- **AI面试训练**：4节系统课（深度版1v1 / 标准版小班+1v1）
- **申请全流程代管**：材料规划 + 系统填报 + 节点提醒
- **督学小助手**：深度版配备陪伴式贴心学姐，全程跟进（详见[督学小助手工作流程](07-工作流程/督学小助手工作流程.md)）

### 第二阶段：复审（校园日）全流程

针对校园日的各项考核环节进行系统训练和全真模拟（深度版10小时1v1）。

核心服务：表达体系建立、群面与互动、批判性思维、学术写作、全真模拟

### 配套附加服务

西交利物浦、宁波诺丁汉、浙大海宁等优质中外合办院校的三位一体报考规划与校测指导。
