# 南桥 SouthBridge Consulting · 业务知识库

## 这是什么？

这是南桥（SouthBridge Consulting）的业务知识库——专注上海纽约大学（NYU Shanghai）和昆山杜克大学（Duke Kunshan University）升学辅导。

**三种读者，一个知识源：**
- 🧑‍💼 **员工 / 合作老师** — 入职 onboarding、了解服务、统一话术
- 🤖 **AI Agent** — 自媒体创作、客服回复、内容运营的上下文注入
- 👤 **管理者** — 团队统一认知，减少重复解释

## 如何使用

1. **所有任务默认加载** `01-公司概览.md`——这是最核心的上下文
2. **需要深入了解某个服务时**，按需加载 `03-服务项目/` 下对应文件
3. **创作内容时** 参考 `05-过往案例/`、`06-品牌调性.md`
4. **新人入职** 按数字顺序阅读 01 → 02 → 03 → 04
