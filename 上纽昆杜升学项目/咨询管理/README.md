## 设计原则

- **按学生归类**：一个学生的所有东西在一个文件夹里——档案、咨询记录、逐字稿、聊天记录。点击学生名字，一切都在里面。
- **跨学生的放外面**：跟进看板（跨学生视图）、话术库（通用资产）、模板
- **档案是活文档**：每次咨询或微信沟通后自动更新
- **聊天记录只增不改**：每个学生一份完整聊天记录，每次上传只追加新消息
