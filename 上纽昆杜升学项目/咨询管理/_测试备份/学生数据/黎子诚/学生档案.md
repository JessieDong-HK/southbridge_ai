# 黎子诚 · 初诊+方案咨询

## 咨询前背景

家长（微信名"蠹"）带学生背景前来咨询。学生来自广东佛山南海外国语高中，成绩优异但学校非省重点。家庭已有清晰的留学预算和方向规划。

## 核心讨论点

### 1. 背景评估：全能型但学校是短板
- **学生画像**：年级前5-10（无掉出记录，高三上升趋势）、学生会主席、哈佛峰会、模联、外研社杯省一/国家二等奖、街舞佛山第一、钢琴十级、邱成桐经济金融建模论文进行中
- **核心问题**：南海外国语非省重点——存在"卡背景"风险
- **我们的评估**：综合背景在非顶尖生源中属于拔尖水平，通过文书包装与面试表现，"进入复试概率非常大"
- **关键洞察**：这是一个"除了学校什么都强"的学生——文书包装是关键中的关键

### 2. 混合申请方案
- **家长需求**：主攻上纽+昆杜，加拿大保底（多伦多/UBC/麦吉尔），英国备选（格拉斯哥/伯明翰），排除澳洲及港新
- **预算**：本硕总计 250-300 万，优先性价比高的中外合办
- **我们的建议**：双线并行——中外合办为主攻，加拿大为保底，英国为备选
- **关键洞察**：家长预算充足且理性——优先 ROI，对中外合办的性价比认可度高，签约阻力小

### 3. 申请策略：无短板原则
- **我们的策略**：
  - 高中背景无法改变 → 必须在语言成绩、活动清单、文书上做到极致
  - 文书需传递与上纽/昆杜匹配的价值观，通过独特包装弥补学校短板
  - 学生会主席+哈佛峰会+邱成桐论文 → 打造"学术+领导力双强"画像
- **学生需求**：最需要帮助的是文书构思和 AI 面试（初审环节），对校园日相对自信
- **关键洞察**：学生知道自己要什么——可推单模块服务（文书+AI面试）降低决策门槛

### 4. 加拿大申请补充
- **我们的建议**：9-12月提交，需高中均分+文书+推荐信+语言成绩
- **费用预估**：多伦多大学四年约 150-200 万（含生活费）
- **关键洞察**：加拿大作为保底方案，学生不需要太多额外辅导——不影响中外合办的签约决策
