# 阎浩翔 · 第一场 · 家长咨询

## 咨询前背景

## 核心讨论点

### 1. 硬指标评估：达标
- **学生背景**：清华附中广华分校，前30%，雅思7.0
- **我们的评估**：硬性指标不存在短板，学校背景有优势，通过初审概率高
- **关键洞察**：家长的关注点是"他行不行"——我们的判断是"硬指标没问题，关键是软实力"

### 2. 活动经历：误区澄清
- **家长表达**：学生仅有 CTB 等少量活动
- **Jess 强调**：活动重质量不重名头——两校更看重学生在活动中展现的潜力与个人特质，而非单纯追求国际大奖
- **示例**：通过深度沟通挖掘生活细节（如帮助外国人指路），将其转化为符合学校偏好的"Gold Story"
- **关键洞察**：家长对"活动"的理解停留在"越多越好"——需要教育家长"一个深度故事胜过十个浅度参与"

### 3. 两校考核体系拆解
- **上纽**：文书（Bridge Builder）→ AI面试 → 校园日（学术圆桌/无领导小组讨论+写作）
- **昆杜**：AI面试 → 校园日（真人面试2对1+写作，类似雅思议论文）
- **录取规则**：A档高考仅需特控线（一本线），B档需综合排名
- **关键洞察**：让家长理解"高考不是唯一标准"+"面试可以大幅拉分"——建立信心

### 4. 内向型面试策略（本场核心）
- **学生特点**：I型人格，公立学校背景，缺面试经验
- **我们的策略**：
  - 心理建设：建立平等对话关系，消除"被考核"的恐惧——"面试是展现自我而非机械答题"
  - 技巧赋能：用 Gold Story 故事库建立表达自信，模拟群面复盘
  - 心态关键：校园日当天必须"豁出去"
- **失败案例分享**：准备时间不足（仅一个月突击）和临场心态失控导致失败
- **关键洞察**：分享失败案例建立了真诚和信任——"我们不保证结果，但我们知道怎么避免失败"

### 5. 暑期规划
- **背景提升**：推荐7月 AI Web Coding + 创业训练营（AI+社会公益 SDG 方向）
- **时间节点**：暑期建立联系+背景挖掘 → 9/10月正式进入课程 → 次年3月公布结果
- **关键洞察**："尽早启动"是促单的核心紧迫感来源

## 咨询前背景

家长场后安排学生与团队沟通。Cherie（上纽A档全奖学姐）主导，目标：深入了解学生动机+背景+性格，同时展示上纽的独特价值。

## 核心讨论点

### 1. 择校动机与专业兴趣
- **学生表达**：倾向主动学习的教育理念，因前两年在国内就读而选择两校；专业方向经济/会计类
- **Cherie 引导**：以校友身份系统介绍上纽——3+1模式、海外学习中心、Stern商学院对接
- **关键洞察**：学生对"为什么选上纽"有自己的想法，不是家长推着走——这是积极信号

### 2. 上纽独特价值展示（本场核心亮点）
- **Cherie 的展示**（校友视角）：
  - Stern 商学院估值课："跟估值界泰斗学习，华尔街都用他的模型"
  - 多元文化环境："一半国际生，70+国家"
  - TEDx NYU Shanghai、迎新周等活动："所有活动都有中国人和外国人一起参与"
- **关键洞察**：Cherie 用亲身经历讲上纽比 Jess 用外部数据讲更有感染力——"过来人"的身份是最好的话术

### 3. 背景活动挖掘
- **学生表达**：
  - CTB 数据分析项目：平台算法对外卖员影响（问卷+访谈→结论：算法缺人文关怀）
  - 皮划艇、篮球、滑雪营等运动经历
- **Cherie 的深度追问**：详细询问了项目主题、团队分工、数据收集方法、结论、团队中领导者的表现
- **关键洞察**：外卖员算法研究项目极其独特——社会关怀+数据分析+批判性思维，是完美的 Gold Story 素材

### 4. 内向性格与面试挑战
- **学生表达**："我性格内向，但内心开放"
- **Cherie 的核心要求**：在小组讨论中必须克服内向、积极发言——"只有高质量或推动进程的发言才会被记录"
- **承诺支持**：提供话题素材库和押题支持——"但开口表达是你必须自己承担的责任"
- **关键洞察**：Cherie 没有说"没关系内向也没事"，而是给了诚实的要求+具体的支持——真诚且可操作

### 5. 两校流程对比
- **Cherie 系统梳理**：
  - 上纽：文书→AI面试→校园日（圆桌会议+学术写作）
  - 昆杜：AI面试→校园日（真人面试+写作），氛围更严肃，中国学生比例更高
- **学生确认**：就初审和入围比例等细节进行了确认
- **关键洞察**：学生问的是细节问题——说明在认真考虑，不是走过场

# 阎浩翔 · 方案沟通 · 2026-06-19

## 咨询前背景

阎浩翔，北京朝阳区，就读于清华附中广华分校，选科语数英+物理+政治+历史，年级排名前30%，雅思7.0，参加过CTB全国赛、帆船比赛、数据分析等活动。父亲（Peter）在第一场家长会中表达了对孩子内向性格的担忧，并希望了解上纽/昆杜的录取标准和申请流程。

本场是学生本人第二场会议（首场学生会议由Cherie主理）。团队在会前做了初步评估，一致认为阎浩翔比父亲描述得更好——有独立想法、活动经历可挖掘，适合冲刺上纽和昆杜。

本场目标：向学生本人介绍服务套餐，解答其对英语口语和专业选择的顾虑，推动决策报名。

## 核心讨论点

### 1. 服务套餐介绍（标准版 vs 深度陪跑版）
- **学生/家长表达**：标准版和深度陪跑版在故事挖掘上有什么区别？如果我在标准版自己准备了答案，有没有机会得到老师的改进建议？
- **我们的回应/建议**：两个版本都有故事深挖环节，但只有陪跑版会一对一帮你把每个答案打磨到成品状态。标准版课后有10-15分钟答疑，可以抢时间问老师，但无法保证深度修改。
- **关键洞察**：学生对服务细节提问精准，说明他在认真评估性价比和自身需求，而非被动接受。这是理性决策型学生，适合用清晰的差异化说明来推动。

### 2. 英语口语能力的顾虑
- **学生/家长表达**：我英语口语不够地道，担心影响申请。
- **我们的回应/建议**：学校更看重你表达的内容，而不是语言是否地道。哪怕有语法错误，只要你的故事有逻辑、有深度，依然能获得青睐。重申打磨好故事内容的重要性远大于语言本身的完美。
- **关键洞察**：英语口语不自信是普高生普遍痛点。学生主动提出这个问题，说明他有自我认知且在意申请质量。Jessie的回应将焦点从"语言能力"转移到"内容深度"，有效降低了学生的焦虑，同时强化了服务价值。

### 3. 团队优势与AI创业营背景提升
- **学生/家长表达**：（未直接提问，在听）
- **我们的回应/建议**：团队老师均有上纽/昆杜A档录取经验，全程亲历申请。重点推荐7月在北京举办的青少年AI创业营——由我们主办，从0到1打造商业项目并产出Demo，属于高含金量的Passion Project，对标藤校申请标准。
- **关键洞察**：AI创业营作为主办方独家资源，是差异化竞争的有力筹码。对处于"刷题空白期"的学生来说，这是一个紧急且高质量的背景提升机会。用"主办方内推"而非"推荐参加"的表述，增强了稀缺感和信任感。

### 4. 择校动机与"Why School"策略
- **学生/家长表达**：我希望聚焦于能提供正反馈且有价值的领域。
- **我们的回应/建议**：以面试常见问题"为什么选择我们学校"为例，展示如何通过匹配学校具体资源（特定教授、演讲系列、社团活动）给出有具体例证的回答，区别于泛泛而谈的"因为学校排名好"。
- **关键洞察**：学生使用"正反馈"和"有价值"这两个关键词，说明他是结果导向型思维，在意投入产出比。Jessie用具体案例展示了如何将泛泛的择校动机转化为说服力强的回答，这个教学方法本身就在展示服务的专业度。

### 5. 专业选择原因（Why Major）
- **学生/家长表达**：选择经济与会计方向，主要是出于性格匹配和家庭资源支持。这个方向的门槛相对较低。
- **我们的回应/建议**：Jessie分享个人经历，并强调回答专业兴趣问题时，需要用个人具体经历来支撑，而非仅仅说"家里有资源"。要说明学校的具体资源将如何支持自己的发展。
- **关键洞察**：学生坦诚地提到家庭资源，但表达方式过于直白（"门槛低"），这在面试中可能减分。Jessie的引导方向是对的——需要把"家里有资源"转化为"我在这个领域已经有认知和体验，学校XX资源能让我进一步XX"。这个转化过程正是服务的核心价值所在。

### 6. 时间紧迫性与行动号召
- **学生/家长表达**：（已基本了解信息，需要与家长商量）
- **我们的回应/建议**：总结两点尽快开始的理由：一是时间紧迫（6月已非早期），可以立即内推参加暑期AI创业营做背景提升；二是需要尽早开始深度挖掘活动经历、打造故事和人物画像，这对接下来的申请起决定性作用。
- **关键洞察**：用"时间窗口正在关闭"来制造紧迫感，同时给出具体的即时行动（内推创业营+开始故事挖掘），让紧迫感不是空泛的压力而是有出口的行动。决策权交给家长，学生作为沟通桥梁。
