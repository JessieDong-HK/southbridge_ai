# 🎙️ AI 面试平台

> 上海纽约大学 & 昆山杜克大学 AI 面试模拟平台
>
> 找开源项目 → 自定义题目和评分 → 给学生做模拟面试

## 快速开始

1. 先读 [AI面试平台项目蓝图.md](./AI面试平台项目蓝图.md)
2. 选型：部署聆悟或 DeepInterview → 跑通全流程 → 决定用哪个
3. 写题库：参考 `题库/` 目录下的维度模板
4. 定评分：参考 `评分标准/` 目录
5. 部署上线

## 目录

| 文件夹 | 用途 |
|--------|------|
| `题库/` | 上纽 + 昆杜的面试题目，按维度分文件 |
| `评分标准/` | 评分维度、权重、追问规则 |
| `workflows/` | Claude Code Workflow（如适用） |
| `prompts/` | AI prompt 模板 |
| `搭建记录/` | 搭建过程记录 |
