# 话术库

> 南桥教育咨询沟通的"弹药库"。统一话术保证底线，个性化话术拉高上限。

## 两层结构

### 01-统一话术（人工维护）
团队共享的标准话术，每个文件一个场景，持续从真实咨询中迭代。

| 文件 | 用途 | 状态 |
|------|------|------|
| `会议话术手册.md` | 完整的咨询会议结构+各阶段话术 | ✅ v1 |
| `开场话术.md` | 各渠道首次接触的标准开场 | ✅ v1 |
| `机构优势话术.md` | "为什么选南桥"的四个核心差异点 | ✅ v1 |
| `FAQ标准回复.md` | 10个高频问题的标准回复 | ✅ v1 |
| `免费评估邀约话术.md` | 免费评估的邀约话术（含各类型变体） | ✅ v1 |
| `价格沟通话术.md` | 报价+价格顾虑应对 | ✅ v1 |

### 02-个性化话术（AI 生成）
为每个学生定制的个性化沟通话术。输入：学生档案 + 咨询记录 + 统一话术。由 `话术引擎` Skill 生成。

## 话术迭代流程

```
真实咨询产生好话术
  → 记录到咨询记录的"我们的话术输出"
  → 标记"建议提升到统一话术库"
  → Jess/Cheryl 审核
  → 更新到对应统一话术文件
  → 标注来源（哪次咨询、对谁说的）
```

## 使用原则

1. **统一话术是参考，不是剧本**——理解背后的逻辑，不要逐字背诵
2. **每个学生的情况不同**——统一话术给的是"说什么"，个性化话术给的是"为什么对他重要"
3. **持续迭代**——每次咨询后发现的好话术，不要让它消失在聊天记录里
